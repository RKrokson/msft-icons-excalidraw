# Routing Rules

| Signal | Route To | Why |
|--------|----------|-----|
| SVG parsing, conversion logic, path handling, element types | Vermeer | Core conversion engine |
| Architecture, pack configs, script structure, dedup strategy | Michelangelo | Design decisions |
| Visual accuracy, test cases, icon comparison, edge cases | Escher | Quality validation |
| README, docs, copilot-instructions, sharing guides | Hokusai | Documentation |
| "Team" or multi-domain | Michelangelo + relevant agents | Cross-cutting |
| Quick factual question | Coordinator (direct) | No spawn needed |
